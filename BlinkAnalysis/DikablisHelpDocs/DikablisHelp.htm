Setting Up

To start streaming data from Dikablis Recorder you must first set up the recorder to the point where you are ready to start recording. This means that you must start the program and calibrate the system for the new patient. Depending on how you have your Dikablis Recorder settings setup you don't necessarily need to start recording. The settings can be found in “settings.xml” located in “C:\Programs\Dikablis\Recorder”. If you open the “settings.xml” file into a text editing program like NotePad you can see the following information.

<settings>
…
  <recorder>
  …
    <remote>
      <enabled>true</enabled>
      <port>2001</port>
      <user>user</user>
      <pass>pass</pass>
      <journal>permanent/recording/never</journal>
      <slaveport>2002</slaveport>
    </remote>
  …
  </recorder>
…
</settings>

This program uses the slave port to receive incoming data. By default the port is 2002. You have three different methods for receiving the incoming data.

  • <journal>permanent</journal>: journal data is always transmitted
  • <journal>recording</journal>: journal data is only transmitted during recording
  • <journal>never</journal>: journal data is never transmitted

For this program to work we recommend that you make sure that you use the “permanent” option.
